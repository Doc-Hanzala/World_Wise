# World Wise

# Features
- Vite App
- CSS Modules
- Context API
- useState
- useEffect

# Libraries
- React Router DOM
- React datePicker
- Leaflet Js
 
# How to Run
- Download the zip File
- Unzip the downloaded file
-  Open in Vs Code
-  Run " npm install " in terminal
-  Run "npm run server " in terminal
-  Run " npm run dev " in terminal
-  Open the link
